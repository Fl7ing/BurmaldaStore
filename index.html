❓ FAQ

Frequently Asked

Everything you need to know about BURMALDA. Can't find your answer? Join our Discord.

What is BURMALDA and what games does it support?
▼
BURMALDA is a premium Roblox utility script featuring over 100 modules including ESP, Combat, Movement, Automation, and Security systems. It is primarily optimized for popular Roblox games but includes universal modules that work across most experiences. We regularly add support for trending games — check our Discord for the latest game compatibility list.
Is BURMALDA free to use?
▼
Yes! BURMALDA is completely free. Simply copy the script from the Download section and execute it with your preferred Roblox executor. We believe powerful tools should be accessible to everyone. Premium tiers with exclusive modules and priority support are planned for the future.
Which executors are compatible with BURMALDA?
▼
BURMALDA has been tested and works with all major Roblox executors including Synapse X, KRNL, Fluxus, Delta, Codex, Hydrogen, and most other Level 7+ executors. We recommend using a reputable executor for the best performance and stability. Some modules may require specific executor capabilities.
Is BURMALDA safe to use? Will I get banned?
▼
BURMALDA includes a built-in anti-detection layer and Safe Position Guard to reduce detection risk. However, using any external script in Roblox carries an inherent risk. We implement obfuscation, velocity normalization, and position validation to minimize footprint. Use responsibly and avoid obvious/aggressive usage in crowded public servers to minimize risk.
How do I execute the BURMALDA script?
▼
1. Open Roblox and join any game. 2. Launch your executor and attach it to Roblox. 3. Copy the script from the Download section above. 4. Paste it into your executor's script box. 5. Click Execute. The BURMALDA menu will appear in-game — press the hotkey shown to open it.
How often is BURMALDA updated?
▼
BURMALDA receives updates approximately every 1-2 weeks. Critical bug fixes and anti-detection patches are deployed within 24-48 hours of issues being reported. Major feature updates are announced in our Discord server. The script auto-updates on each execution — no manual re-download required.
What is Silent Aim and how does it work?
▼
Silent Aim redirects your projectiles toward the nearest valid target without visually moving your cursor. Unlike traditional aimbot, Silent Aim is virtually undetectable by other players as your cursor appears to aim normally. It uses prediction algorithms to compensate for target movement and network latency for maximum accuracy.
Can I customize the ESP colors and settings?
▼
Absolutely! BURMALDA features a comprehensive color customization system. You can set custom colors for each ESP type individually, enable team-based coloring, toggle visibility-based color changes, and even use the Rainbow Highlight mode for dynamic color cycling. All settings are saved automatically between sessions.
Does BURMALDA work on mobile (Android)?
▼
Yes, BURMALDA has an Android-compatible version optimized for mobile devices. The mobile version features a touch-friendly interface, reduced performance overhead, and all core ESP and automation modules. Some advanced combat modules are exclusive to PC. Download the Android version from the Download section above.
The script isn't working. What should I do?
▼
First, ensure your executor is properly attached to Roblox and is up to date. Try clearing your executor's cache and re-executing the script. Check our Discord #status channel to see if there are any known issues. If the problem persists, copy the error message from your executor console and post it in our #support channel for help.
How do I open the BURMALDA menu in-game?
▼
After executing the script, press the INSERT key (default) to toggle the main menu. You can also right-click anywhere on screen to open a quick-access menu. All keybinds are fully customizable from the Interface settings tab within the hub. The menu is draggable and can be repositioned anywhere on screen.
Does the FPS Boost module actually improve performance?
▼
Yes! The FPS Boost module disables unnecessary client-side rendering elements, reduces particle effects, optimizes texture streaming, and applies memory cleanup routines. Users typically see a 30-60% increase in frame rate depending on their hardware. Combined with Performance Mode, this can dramatically improve gameplay smoothness on lower-end devices.
Can I contribute to BURMALDA development?
▼
Yes! BURMALDA is open-source and welcomes contributions. Visit our GitHub repository to submit pull requests, report bugs, or suggest new features. Community-developed modules are reviewed and may be integrated into the main release. Join our Discord's #developer channel to coordinate with our dev team and get API documentation.
What is the Coin Farm module and is it safe?
▼
The Coin Farm module automates the collection of in-game currency by detecting and pathing toward collectible items while maintaining safe movement patterns. It uses Smart Follow algorithms to appear as natural player behavior. The Anti AFK module works alongside it to prevent disconnection. Use with reasonable collection speeds to avoid detection by game-specific anti-cheat systems.
Where can I get support and report bugs?
▼
The best place to get support is our Discord server — join via the link in the footer. We have a dedicated #support channel with active moderators and developers. For bug reports, use our GitHub Issues page with detailed reproduction steps. Response times are typically under 2 hours during active server hours.
